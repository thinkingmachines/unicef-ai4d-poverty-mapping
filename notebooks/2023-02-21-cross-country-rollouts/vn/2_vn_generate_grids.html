Generate Roll-out Grids: Vietnam

This notebook generates the rollout grid tiles over the country. The output file is used as an input for Step 3, where we will run the trained model over the set of grids.

The generated grids are set at 2.4km (zoom level 14), matching the grids used during model training. The grids are also filtered to only include populated areas based on Metaâ€™s High Resolution Settlement Layer (HRSL) data.
